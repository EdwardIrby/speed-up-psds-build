Installation
Buttons

Buttons

Install the component dependency:

npm install @pluralsight/ps-button --save-dev

Include a React component in your project:

import Button from '@pluralsight/ps-button/react'

For more project setup guidance, see the Component Installation Instructions.

Button appearance

Define a button appearance by ... 

import Button from '@pluralsight/ps-button/react'

<Button>Click me</Button>
<Button appearance="stroke">Click me</Button>
<Button appearance="flat">Click me</Button>

Button sizes

import Button from '@pluralsight/ps-button/react'

<Button size="large">Click me</Button>
<Button size="medium">Click me</Button>
<Button size="small">Click me</Button>
<Button size="tiny">Click me</Button>

Disabled button

import Button from '@pluralsight/ps-button/react'

<Button disabled="true">Disabled button</Button>
<Button disabled="true" appearance="flat">Disabled button</Button>
